Java中List add方法数据被覆盖问题解决

发表于 2020-04-03 分类于 Java

问题描述

List<List< Integer >> add(int index, List< Integer>)添加新的List时，会覆盖之前已有的数据

解决方案

将内部List初始化放在循环体内部

原因分析

add()方法添加的是引用，如果上述一行代码在循环外面，那么创建的引用始终只有一个。
而如果将上述一句代码放入循环内，则相当于每次循环都创建了一个引用，并且指向当次循环的List。
