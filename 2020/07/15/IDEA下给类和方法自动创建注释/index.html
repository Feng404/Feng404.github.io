IDEA下给类和方法自动创建注释

发表于 2020-07-15 分类于 IDE技巧

生成类时自动生成注释

右击File打开Settings或者使用快捷键(Ctrl+Alt+S)

打开Settings后，在输入框搜索File and Code Template

在Files -> Class 内部添加注释模板
点击Apply或者OK即可生效，Interface类似

添加方法注释

进入Settings后在搜索框内输入Live Template

点击右方加号，选择添加Template Group
在弹出group name 输入框内输入名称

点击新建立的Template Group MyNewDefine，再次点击右方加号，选择Live Template

添加完空模板的样子

1 简写
2 模板说明
3方法注释模板
4是设置触发条件，可以使Enter Tab Space，默认是Tab，我设置的是Enter

设置参数

点击Edit variables 在弹出框内添加Default value

输入的信息

点击下方Change选择Java为模板应用的文件类型
点击Apply 或者OK即可生效
在方法体上输入 /**enter 即可自动生成方法注释
